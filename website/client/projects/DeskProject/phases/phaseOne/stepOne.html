System Architecture Design

In this step, we finalized the core system architecture that will enable flexible AV control without the complexity and cost of a full matrix switcher.

The Problem

Traditional setups require unplugging cables to switch devices between monitors. For gaming, work, and laptop use, this becomes tedious with multiple monitors.

The Solution

Use 3 independent HDMI switches (one per monitor) combined with a splitter for the Xbox. This allows each monitor to independently select its input source.

Why Not a Matrix?

While a 4×4 HDMI matrix would work, it costs significantly more and adds unnecessary complexity for our needs. The switch-based approach achieves the same functionality at a fraction of the cost.

Hardware Decisions

  • HDMI Switches: 3-port switches for each monitor (Desktop, Laptop, Xbox inputs)
  • Xbox Splitter: 1×4 HDMI splitter to feed the Xbox signal to all switches
  • Laptop Dock: USB-C dock with 3 HDMI outputs
  • Desktop Adapters: DisplayPort to HDMI for additional desktop outputs

Wiring Overview

Total: 11 HDMI cables. Each switch receives inputs from Desktop, Laptop, and Xbox (via splitter), then outputs directly to its monitor.

This architecture provides true flexibility: any device on any monitor, independently, without cable swapping.
