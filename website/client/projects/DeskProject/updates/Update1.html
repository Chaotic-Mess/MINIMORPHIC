2026 Desk Project / Planning Phase

Planning Phase: System Architecture & Design

Welcome to the 2026 Desk Project devlog. This is where I document the creation of an L-shaped electric sit/stand desk with advanced AV control for three monitors, Xbox, desktop, and laptop. The goal is flexible device-to-monitor routing without constant cable unplugging.

Goal

Create an L-shaped electric sit/stand desk for the bedroom with clean cable management and flexible AV control for three monitors, an Xbox, a desktop, and a laptop.

Primary Objective: Allow any device to appear on any monitor without unplugging cables.

Examples of Desired Functionality

  • Xbox on center screen, desktop on left/right
  • Laptop on all three screens
  • Desktop on all three
  • Any mix, per monitor

Final System Architecture (Locked)

This project will NOT use a 4×4 matrix. Instead, the system uses:

  • 3 independent HDMI switches (one per monitor)
  • 1 HDMI splitter for the Xbox
  • 1 triple-HDMI laptop dock
  • Desktop connected using HDMI + DisplayPort→HDMI adapters

This achieves the same functionality as a matrix at a much lower cost and with simpler setup.

Core Concept

Each monitor gets its own input selector. Each monitor can independently choose between Desktop, Laptop, or Xbox.

Hardware Layout

Monitors:

3 total monitors.

Desktop:

  • 1 HDMI output
  • 3 DisplayPort outputs
  • Use: 1 HDMI → Switch 1, 2 DisplayPort → HDMI adapters → Switch 2 & 3

Laptop:

  • USB-C dock providing 3 HDMI outputs
  • Use: HDMI 1 → Switch 1, HDMI 2 → Switch 2, HDMI 3 → Switch 3

Xbox:

  • 1 HDMI output
  • Use: HDMI → 1×4 splitter → feeds all 3 switches

Switches:

Each monitor has one 3-input HDMI switch.

Inputs to each switch:

  • Input 1 → Desktop
  • Input 2 → Laptop
  • Input 3 → Xbox (from splitter)

Output: Goes directly to that monitor.

Physical Wiring Map

Switch 1 (Monitor 1):

  • Desktop HDMI
  • Laptop HDMI #1
  • Xbox splitter output #1

Switch 2 (Monitor 2):

  • Desktop DP→HDMI adapter #1
  • Laptop HDMI #2
  • Xbox splitter output #2

Switch 3 (Monitor 3):

  • Desktop DP→HDMI adapter #2
  • Laptop HDMI #3
  • Xbox splitter output #3

Functionality Achieved

This system allows:

  • Gaming mode: Center → Xbox, Left/right → Desktop
  • Work mode: All screens → Desktop
  • Laptop mode: All screens → Laptop
  • Mixed mode: Any monitor can independently display any device

Performance Target

Baseline spec: HDMI 2.0 class switching, 4K @ 60Hz stable, HDR supported (depending on switch quality).

Note: This design prioritizes flexibility and cost over 4K120 routing. If high-refresh gaming is needed later, the desktop can still be connected directly via DisplayPort to one monitor as a future upgrade.

Future Expansion (Phase 2 – Optional)

Start manual using switch buttons. Later upgrade possible by:

  • Adding Arduino control to press switch buttons
  • Adding a touchscreen interface under the monitors
  • Adding presets: Gaming, Work, Laptop, Mixed

Display Screen Concept (Future)

Small display panel under monitors showing:

  • Monitor 1: Desktop
  • Monitor 2: Xbox
  • Monitor 3: Laptop

Possible later features: Logo icons, Transition text, Status indicators.

Desk Build Goals

  • L-shaped electric sit/stand desk
  • Hidden cable routing
  • Under-desk mounted switches + splitter
  • Minimal visible wiring
  • Wireless keyboard/mouse
  • Headset + controller stand
  • Clean aesthetic

Design Philosophy

  • Keep wiring simple
  • Avoid over-engineering early
  • Avoid expensive AV matrices
  • Modular so parts can be upgraded later

DisplayPort Note

Originally considered converting everything to DisplayPort. Updated decision: Keep switching backbone in HDMI for compatibility and cost.

Reason: Xbox is HDMI only, Laptop docks output HDMI, HDMI switches and splitters are far more available and affordable. Desktop still benefits from native DisplayPort outputs and can be adapted to HDMI for switching.

Final Direction Locked

No matrix. No custom AV routing hardware. No complex electronics in phase 1.

System will use: Splitter + 3 switches, Clean wiring, Modular upgrade path later.
